Step 1:

Install a Userscript Manager such as ViolentMonkey or TamperMonkey for your Browser. The links in the table below will take you to the appropriate Web App Store for your browser. In most cases it is a one step process (Opera requires two addons to be installed). Click the appropriate button to install the addon and then click accept in the dropdown. Once you have added the userscript manager come back to this page to install the PebblePad userscripts in step 2.

Step 2:

Install the PebblePad modification scripts. You can install them one are a time, or everything at once. Click the link to install the script and then click "Confirm Installation" at the top of the screen. This scripts are dependent on the PebblePad source code, and so PebblePad updates are likely to break the scripts - I will post updates here when that happens.
